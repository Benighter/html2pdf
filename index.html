About This App

This application converts HTML code to PDF using the HTML2PDF API. For best results:

  • Include complete HTML structure with <html>, <head>, and <body> tags
  • Use standard CSS that works in most browsers
  • For custom styling, include CSS within <style> tags
  • External resources like images should use absolute URLs

Tip: Use the "Load Sample" button to see an example of properly formatted HTML.
